## Getting Started

1.  Install Node.js
2.  Familiarise yourself with [Vue.js](https://vuejs.org/)
3.  Clone project and `cd` into project folder
4.  Run `npm install` to install dependencies

Run `npm run serve` to serve files on http://localhost:8080
